n8n Blog Workflows

Right-click → Save link as…, then in n8n cloud: Workflows → Import from File.

1. Publish Blog

Webhook → reads {slug}.json from your Drive folder → publishes to /api/blogs → pings Google + IndexNow.

Becomes: POST https://palmbaylotsland.app.n8n.cloud/webhook/publish-blog

Download publish-blog.workflow.json

2. Doc → JSON Draft (Helper)

Converts a structured Google Doc into a {slug}.json in your Drive folder, ready for the publish workflow.

Becomes: POST https://palmbaylotsland.app.n8n.cloud/webhook/doc-to-json

Download doc-to-json.workflow.json
