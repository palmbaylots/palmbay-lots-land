# How to Update Social Media Links

## File Location
`/app/frontend/src/components/SocialShareButtons.jsx`

## What to Update

1. **Line 22-27: Social Profile URLs**
   Replace these with your actual social media profile URLs:
   
   ```javascript
   const socialProfiles = {
     facebook: 'https://www.facebook.com/YOUR_PROFILE_HERE',
     linkedin: 'https://www.linkedin.com/in/YOUR_PROFILE_HERE', 
     instagram: 'https://www.instagram.com/YOUR_PROFILE_HERE',
     tiktok: 'https://www.tiktok.com/@YOUR_PROFILE_HERE'
   };
   ```

2. **Save the file** - The changes will appear immediately on your website

## Example
If your Facebook page is: https://www.facebook.com/vahidrealestate
Change line 23 to:
```javascript
facebook: 'https://www.facebook.com/vahidrealestate',
```

That's it! The Follow buttons will now link to your actual profiles.
